One Click Build System

Project Info:

The One Click Build System is a procedural building tool designed to assist 3D modelers in creating structures ranging from small-scale buildings to expansive cityscapes with just a few clicks and toggles. This versatile system is available in two versions: one tailored for Unreal and another for Unity.

Project Details:

  • Technologies:C++, Unreal built-in libraries, UE Blueprint
  • Date:January 2021 - June 2021
  • Reference: Assassin's Creed Build System

Multi-Bone IK

Project Info:

Multi-Bone IK is a user-friendly system designed to streamline the complex animation development process in games, movies, and virtual worlds. Developed within the Unreal Engine (UE) framework, this system caters to various targets including bipedal, quadpedal, eight-legged (spider), and legless (snake) entities. Moreover, the IK system extends beyond the traditional application to feet, encompassing precise control over the head, neck, spine, elbows, knees, hands, and even fingers.

Project Details:

  • Technologies:C++, Unreal Engine, Python
  • Date:Jan. 2021 - May 2021
  • Reference: Dragon IK

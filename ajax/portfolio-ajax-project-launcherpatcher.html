Launcher Patcher System

Project Info:

The Launcher-Patcher system was developed for the N1 project development team to streamline the daily build process. Leveraging Jenkins, we effectively manage the game server for the N1 project. The system automates the entire process, handling updates, compression, and deployment of both the server and client components.

Project Details:

  • Technologies: C#, Jenkins, .Net Core
  • Date:December 2020

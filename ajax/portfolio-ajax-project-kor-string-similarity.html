One Click Build System

Project Info:

One Click Build System is a procedural building system to aid 3D modelers to build from very small unit like a building to a very large unit, like a city. The system is built in two versions - Unreal and Unity.

Project Details:

  • Technologies:C++, Unreal built-in libraries, UE Blueprint
  • Date:January 2021 - June 2021
  • Reference: NPM

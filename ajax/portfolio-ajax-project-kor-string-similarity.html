kor-string-similarity

Project Info:

kor-string-similarity is an npm module that calculates string similarity between two strings(Korean, alphabet, number, special characters, etc.), based on dice coefficient and NphoEd(Normallized phoneme-based metric).

Project Details:

  • Date:June 2018 - July 2018

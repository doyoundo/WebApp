N1

Project Info:

N1 is a game project launched by Vespa Interactive. The game is a AAA MMORPG game that can be played on PC and console. The game has been developed for 5 years, and the project has been elected three times for innovation and new technology implementation awards. The server is designed with AWS, and the client side is manily developed with Unreal Engine.

Project Details:

  • Technologies:AWS, Unreal Engine, C++, Python, Cross Platform Implementation
  • Date:June, 2020 - Feb. 2022

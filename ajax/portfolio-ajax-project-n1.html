N1

Project Info:

N1 stands as a flagship AAA game project unveiled by Vespa Interactive, offering an immersive MMORPG experience available on both PC and console platforms. The game underwent a meticulous 5-year development cycle, garnering recognition for innovation and pioneering technology implementation—specifically in graphic optimization and computer animation—for three consecutive years. The server architecture is meticulously crafted using AWS, while the client-side experience is predominantly crafted within the robust framework of Unreal Engine.

Project Details:

  • Technologies:AWS, Unreal Engine, C++, Python, Cross Platform Implementation
  • Date:June, 2020 - Feb. 2022

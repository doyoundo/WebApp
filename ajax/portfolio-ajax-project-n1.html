N1

Project Info:

N1 is a game project launched by Vespa Interactive. The game is a AAA MMORPG game that can be played on both PC and console. The game was developed for 5 years, and the project was elected three years in a row for innovation and new technology (graphic optimization and computer animation) implementation awards. The server is designed with AWS, and the client side is mainly developed with Unreal Engine.

Project Details:

  • Technologies:AWS, Unreal Engine, C++, Python, Cross Platform Implementation
  • Date:June, 2020 - Feb. 2022

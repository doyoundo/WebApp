CS2340 Course Website

Project Info:

This is the course website for Georgia Tech CS2340(Object & Design) of which I was in charge of leading the UI renovation project.
The page is built using Jekyll of Ruby on Rails, HTML/CSS/Javascript, and markdown. I implemented Bootstrap-based UI system and created custom components including toggle dropdown and multi-level nav bar.

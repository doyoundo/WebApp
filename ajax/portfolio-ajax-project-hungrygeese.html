Hungry Geese Competition

Project Info:

Hungry Geese, a multi-player simulation on Kaggle, involves training an AI agent to compete against other 'geese' with the goal of being the last surviving player. The agent is programmed to continuously search for food while avoiding collisions with other agents. Throughout this project, I conducted training sessions using various deep reinforcement learning algorithms and neural network architectures, aiming to achieve the highest possible score for the geese

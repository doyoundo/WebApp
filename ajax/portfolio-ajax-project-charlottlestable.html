Charlotte's Table

Project Info:

Charlotte's Table, launched in August 2022, is a puzzle game where, as the AI project manager, I spearheaded the transformation of its original extensive source code into an RL (Reinforcement Learning) simulator. This simulator enables the training of agents, facilitates the collection of observation/action commands, and offers diverse tactics tailored for various puzzle sets.

Project Details:

  • Technologies:C#, Unity, Python, GCP, Pytorch, Ray
  • Date:March 2022 - July 2023

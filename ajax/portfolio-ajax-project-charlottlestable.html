RL Balancing on Charlotte's Table

Project Info:

Charlotte's Table is a puzzle game launched in August 2022. As a project manager for this game, I modified the game code for a RL agent. The agent trained in the modified game environment, collected data, and came up with different tactics in each map and gimmicks. The training and evaluate processes are done on Google Cloud Platorm with 5000 virtual machines for a week.
